# 🐱 냥이 매치 - 고양이 품종 맞춤 추천 서비스 기획서

---

## 1. 서비스 개요

### 1.1 서비스 명
**"냥이 매치"** (가제)

### 1.2 서비스 컨셉
사용자의 라이프스타일, 성향, 환경을 분석하여 가장 잘 어울리는 고양이 품종을 추천하는 MBTI 스타일 테스트 서비스

### 1.3 서비스 목표
- 예비 집사들의 고양이 품종 선택 어려움 해소
- 입양 후 조기 반려 포기율 감소
- 광고 수익을 통한 수익 창출

### 1.4 서비스 형태
- 단발성 테스트 서비스 (백엔드 없는 스태틱 웹사이트)
- 진입 → 테스트 → 결과 → 공유의 간단한 흐름

---

## 2. 타겟 시장

### 2.1 주요 타겟
- 2030 여성 직장인 (고양이 입양 1순위 계층)
- 1인 가구 거주자
- 반려동물 초보자
- 반려동물 관심 높은 SNS 유저

### 2.2 한국 시장 특성
- **반려동물 시장 규모**: 4조원 (2024 기준)
- **고양이 선호도**: 70%가 고양이 선호
- **입양 경로 실태**: 60%가 인터넷 정보만으로 품종 결정
- **MBTI 테스트 열풍**: 간단한 테스트 결과 공유 문화 형성

## 3. 서비스 기능

### 3.1 핵심 기능

**① 냥이 테스트 (MBTI 스타일)**
- 테스트 형식: 12-15개 객관식 질문
- 카테고리: 라이프스타일, 성향, 선호도
- 소요 시간: 1분 내외
- **결과**: 매칭 점수 기반 상위 3개 품종 추천

**② 결과 페이지**
- 매칭된 품종 프로필 (성격, 관리 난이도, 적합 환경)
- 품종별 매칭 점수 (백분율)
- **다양한 결과 이미지 생성** (인스타 스토리, 스레드, 피드용)
- 관련 추천 품종
- 친구 결과 비교 기능

**③ SNS 공유 (바이럴 최적화)**
- 인스타그램 스토리 공유 (세로 이미지 자동 생성)
- 스레드 공유 (최적 가로세로 비율)
- 카카오톡 공유 (미리보기 썸네일)
- 이미지 저장 (앨범에 저장 후 직접 공유)
- 링크 복사
- 결과 URL 직접 공유 (파라미터 기반)
- 공감형/도발형 공유 카피 자동 생성

**④ 인기 품종 랭킹**
- 한국 인기 고양이 품종 순위
- 간단한 품종별 정보 카드

### 3.2 광고 배치

**애드센스 배치 전략**
- **메인 페이지**: 상단 배너 (728x90)
- **테스트 진행 중**: 사이드 배너 (300x250) - 모바일 제외
- **결과 페이지**: 상단 + 하단 배너
- **공유 페이지**: 네이티브 광고

## 4. 매칭 알고리즘 (클라이언트 사이드)

### 4.1 평가 항목 (5대 카테고리)

| 카테고리 | 항목 | 가중치 |
|----------|------|--------|
| 라이프스타일 | 외출 빈도, 주거 공간, 혼사 유무 | 30% |
| 성격 | 활동성, 애정 표현, 사교성 | 25% |
| 관리 용이성 | 그루밍 빈도, 훈련 난이도 | 20% |
| 비용 예산 | 초기 비용, 월평균 비용 | 15% |
| 외형 선호 | 크기, 털 길이, 색상 | 10% |

### 4.2 매칭 로직 (JavaScript)
- 각 질문별 선택지에 점수 부여
- 5대 카테고리별 점수 집계
- 가중치 적용 후 최종 점수 계산
- 상위 3개 품종 추천

## 7. 기술 스택

### 7.1 프론트엔드
- **프레임워크**: Next.js (React) - SSG/SSR 지원
- **스타일링**: Tailwind CSS
- **상태 관리**: React Context API (간단한 테스트 상태)
- **배포**: Vercel (무료 플랜 가능)

### 7.2 광고
- **Google AdSense**: 반응형 광고
- **GTM (Google Tag Manager)**: 광고 트래킹

### 7.3 추가 라이브러리
- **Framer Motion**: 부드러운 애니메이션
- **html2canvas**: 결과 이미지 생성 (인스타 스토리, 스레드)
- **kakao.js**: 카카오톡 공유
- **react-share**: 페이스북, 트위터 공유
- **next-seo**: OG 태그 관리
- **confetti**: 결과 발표 시 효과

### 7.4 백엔드 불필요
- 모든 로직 클라이언트 사이드 처리
- 품종 데이터 JSON 파일로 관리
- GA (Google Analytics)를 통한 트래킹
