# Development

## Requirements
- Node.js (권장: LTS)
- npm

## Install
```bash
npm install
```

## Run
```bash
npm run dev
```

## Build
```bash
npm run build
npm run start
```

## Notes
- `next build` 시 GA 측정 ID가 없으면 경고 로그가 뜰 수 있습니다. (기능상 문제는 없음)
- 워크스페이스에 lockfile이 여러 개 있으면 Next.js가 루트 경로를 경고할 수 있습니다.
