## 5. localStorage 사용 계획

신규 기능으로 추가될 localStorage 키:

```typescript
// 출석 보상
interface AttendanceData {
  lastVisitDate: string;
  currentStreak: number;
  totalVisits: number;
  claimedRewards: number[];
  monthlyAttendance: string[];
}
const ATTENDANCE_KEY = 'nyongmatch_attendance';

// 수집 카드
interface CardCollection {
  collectedBreedIds: string[];
  cardRarities: Record<string, 'common' | 'rare' | 'epic' | 'legendary'>;
  totalCardsFound: number;
}
const CARD_KEY = 'nyongmatch_cards';

// 뽑기 이벤트
interface LuckyDrawData {
  lastDrawDate: string;
  currentStreak: number;
  inventory: string[];
}
const LUCKY_KEY = 'nyongmatch_lucky';

// 명예의 전당
interface HallOfFame {
  highestScore: { score: number; breedId: string; date: string };
  longestStreak: { days: number; endDate: string };
  mostBreedsDiscovered: { count: number; date: string };
}
const HALL_KEY = 'nyongmatch_hall_of_fame';

// 숨겨진 퀘스트
interface QuestProgress {
  unlockedHiddenBreeds: string[];
  completedQuests: string[];
}
const QUEST_KEY = 'nyongmatch_quests';
```

## 7. 결론

DB 없이 localStorage와 정적 JSON만으로 구현 가능한 15개 신규 기능 아이디어를 제안했습니다.

**핵심 추천:**
1. **출석 보상 + 뽑기** → DAU 증가의 핵심
2. **수집 카드 시스템** → 장기 리텐션 핵심
3. **동적 OG + 유명인 카드** → 바이럴 확산 핵심
4. **FAQ SEO** → 검색 유입 핵심

이 기능들은 모두 기존 아키텍처를 유지하면서 점진적으로 추가 가능합니다.
